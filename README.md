# 🍽️ CraveFeed

CraveFeed is a MERN stack-based web application for food discovery and restaurant interaction. It implements core backend concepts including middleware, REST APIs, authentication using JWT and bcrypt, session handling, and MongoDB integration, with role-based access for users and restaurants.

---

## 🚀 Features

- 🔐 User Authentication (Register/Login)
- 🧑‍🍳 Role-based system (User & Restaurant)
- 🔑 Password hashing using bcrypt
- 🎟️ JWT-based authentication
- 📦 REST API using Express.js
- 🗄️ MongoDB database integration (Mongoose)
- ⚙️ Middleware handling

---

## 🛠️ Tech Stack

- Frontend: HTML, CSS, JavaScript  
- Backend: Node.js, Express.js  
- Database: MongoDB Atlas  
- Authentication: JWT, Bcrypt

## ⚙️ Setup Instructions

1. Clone the repository  
git clone <repo-link>

2. Install dependencies  
npm install

3. Add MongoDB connection string in `server.js`

4. Run the server  
node server.js
